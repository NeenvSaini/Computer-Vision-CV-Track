# Computer-Vision-CV-Track
The OpenCV Zoo is a collection of pretrained deep learning models (no training needed) that you can directly use for tasks like:  Face detection, Object detection, Pose estimation, Image classification. Right now this only have face detection soon i will add hand and finger detection, where by using finger detection we can play some sort of sound.

I chose this compuuter vision as i always wanted to do something related to body detection, and i can work on it in future too, like using finger detection to notice the changes in the finger and play a certain note.

Approach:- 
The project uses a pretrained OpenCV Zoo model (YuNet) for face detection. The model is loaded using OpenCV, input is provided through webcam, and inference is performed to detect faces. The results are displayed by drawing bounding boxes on the detected faces.

difficulties faced:- debugging the program, installing the onnx file.

resolutions of the difficulties:- Used AI models like claude to understand and debug the code,

Results:- Face can be now detected thorugh webcam

Learnings :- grasped the concept of opencv and detection via python. 

how to run:- Run the videoface file and face detection should work.
